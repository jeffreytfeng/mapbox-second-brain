## How to engage in a doc review

- Specifics over slogans. "Replace HERE for BMW by Q2 2026" beats "achieve victory with BMW."
- Connect every initiative to revenue. If she can't connect the dots, she'll say so directly.
- A goal without a named pipeline is a hope, not a plan. Show `pipeline ≥ revenue_target / win_rate`.
- For any incremental headcount: state explicitly how AI tools were leveraged to try to avoid the hire. This is non-negotiable.
- Pre-align cross-team dependencies BEFORE the review. "After-OP" alignment doesn't count.
- Frame technical work in business outcomes (compliance, OEM retention, deal protection).
- Reconcile against last year's OP1 — performance assessment builds credibility.
